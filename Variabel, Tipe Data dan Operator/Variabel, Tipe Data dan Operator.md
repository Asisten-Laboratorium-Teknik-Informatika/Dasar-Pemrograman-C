## Variabel, Tipe Data dan Operator



### Variabel
 

1. Variabel dalam pemrograman adalah nama simbolis atau wadah yang digunakan untuk menyimpan data dalam memori komputer, yang nilainya bisa berubah-ubah selama eksekusi program.



2. Variabel berfungsi seperti label pada sebuah kotak, di mana label tersebut adalah nama variabel, dan isi kotak adalah data yang bisa berupa angka, teks, atau tipe data lainnya.



3. Penggunaan variabel memudahkan programmer untuk menyimpan, mengakses, dan memanipulasi informasi dalam program tanpa harus mengingat serangkaian data secara langsung .
 



### Tipe Data



1. Tipe data adalah atribut yang terkait dengan sepotong data yang bisa memberi tahu sistem komputer cara untuk menafsirkan nilainya.


2. Integer: Tipe data bilangan bulat. Program dengan tipe data ini tidak mendukung penggunaan huruf maupun pecahan desimal.


3. Float: Tipe data angka yang memiliki bagian desimal di akhir angka (pecahan).

4. Double: Sama halnya dengan float yang menyatakan bilangan pecahan, namun kapasitas penyimpanannya lebih besar daripada float.


5. Character: Digunakan untuk menyimpan satu karakter tunggal seperti huruf, angka, simbol, atau spasi, yang diapit oleh tanda kutip tunggal.

6. Const: Variabel dengan tipe data const nilainya tidak dapat diubah.


### Format Specifier

1. Format specifier adalah simbol yang digunakan dalam bahasa pemrograman C untuk menentukan tipe data yang akan diinput atau dicetak, menggunakan simbol `%`.

2. `%c`: Digunakan untuk menampilkan karakter .

3. `%s`: Digunakan untuk menampilkan teks (string).


4. `%d`, `%i`: Digunakan untuk menampilkan bilangan desimal.


5. `%f`: Digunakan untuk menampilkan bilangan pecahan.

### Operator


1. Operator adalah simbol atau kata khusus yang digunakan untuk melakukan operasi pada satu atau lebih nilai (dikenal sebagai operand).


2. Operator Aritmatika: Operator yang digunakan dalam melakukan suatu proses operasi matematika, seperti penjumlahan, pengurangan, perkalian, pembagian, dan sisa hasil bagi.

3. Operator Increment \& Decrement: Operator increment digunakan untuk menambah variabel sebanyak satu angka, sedangkan decrement untuk mengurangi variabel sebanyak satu angka.


4. Operator Perbandingan: Dipakai untuk membandingkan 2 buah nilai, apakah nilai tersebut sama besar, lebih kecil, atau lebih besar. Hasilnya adalah 1 (True) atau 0 (False).

5. Operator Logika: Digunakan untuk operasi logika Boolean, seperti AND (`\&\&`), OR (`||`), dan NOT (`!`) \[cite: 162].
